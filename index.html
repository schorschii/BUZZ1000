BUZZ1000 Webapps

Arduino Controlled Buzzer System For Events

Please connect the system to your computer and open a buzzer app:

  • Simple Buzzer App
  • Buzzer App With Bars
  • Buzzer App With Timer

How to play offline

  1. Go to the GitHub repo, click on "Code" -> "Download ZIP"
  2. Extract the ZIP file onto your desktop
  3. Open index.html with Chrome or Firefox

Please visit the Github repository for more information.
